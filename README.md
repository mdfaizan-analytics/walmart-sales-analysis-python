# 🛒 Walmart Sales Data Analysis using Python

## 📌 Project Overview
This project analyzes Walmart sales data using Python to uncover business insights and demonstrate essential data analysis skills. The project covers data cleaning, exploratory data analysis (EDA), and data visualization to identify meaningful sales trends.

## 🎯 Objectives
- Clean and preprocess the dataset
- Explore the data using Python
- Perform exploratory data analysis (EDA)
- Visualize sales trends and patterns
- Generate business insights from the data

## 🛠️ Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib

## 📂 Dataset
- **Dataset:** Walmart Sales Dataset
- **Source:** Kaggle

## 📊 Project Workflow
1. Import the dataset
2. Explore the dataset
3. Check data types
4. Handle missing values
5. Remove duplicate records
6. Perform data cleaning
7. Generate summary statistics
8. Conduct Exploratory Data Analysis (EDA)
9. Create visualizations
10. Draw business insights

## 📈 Key Analysis
- Dataset overview
- Missing value analysis
- Duplicate value check
- Statistical summary
- Sales trend analysis
- Category-wise analysis
- Data visualization using Matplotlib
